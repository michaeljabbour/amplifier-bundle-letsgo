The Architecture

Six satellite bundles

Each bundle owns one concern. No overlap, no bloat.

Core

Gateway daemon, security, secrets management, memory, telemetry, onboarding wizard

Voice

Auto-transcribe audio, optional TTS. Whisper API, local Whisper, edge-tts, ElevenLabs, OpenAI

Canvas

Agent-driven visual workspace. Vega-Lite charts, HTML, SVG, markdown, code, tables

Browser

Web automation via browser-tester. Operator, researcher, visual-documenter agents

WebChat

Chat UI + 6-tab admin dashboard. Sessions, channels, senders, cron, usage, agents

MCP

Bridge to external MCP tool servers. Single mcp_call tool with server/tool routing

Gateway

13 channel adapters

Plugin system with Python entry-point discovery. Install and go.

Built-in

Webhook Discord Telegram Slack WhatsApp

The three-layer canvas

Get Started

Try it today

Three steps to your own multi-channel AI assistant.

# 1. Include in your bundle
bundle:
  name: my-assistant
includes:
  - amplifier-bundle-letsgo
  - amplifier-bundle-letsgo-voice
  - amplifier-bundle-letsgo-canvas

# 2. Run the setup wizard
recipes execute letsgo:recipes/setup-wizard.yaml

# 3. Start your assistant
amplifier start
